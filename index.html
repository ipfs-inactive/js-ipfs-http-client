Intro

Installable via npm install --save ipfs-http-client, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save ipfs-http-client

You can then require() ipfs-http-client as normal:

const ipfsHttpClient = require('ipfs-http-client')

In the Browser

Ipfs-http-client should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of ipfs-http-client, including index.js and index.min.js, are included in the /dist folder. Ipfs-http-client can also be found on unpkg.com under

  • https://unpkg.com/ipfs-http-client/dist/index.min.js
  • https://unpkg.com/ipfs-http-client/dist/index.js
